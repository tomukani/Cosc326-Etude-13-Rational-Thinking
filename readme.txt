To run the program

navigate to directory and type

compiles to c++98 by default so set compiler to the latest standard
g++ *.cpp -std=c++0x -o rational

to run
./rational

to feed a file into the executable
./rational < test.txt
The program will ask for two numbers

We used this wikipedia page to help with implementing Rational arithmetic
https://en.wikipedia.org/wiki/Rational_number#Arithmetic
